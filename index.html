🚀 Getting Started

          

            
The Mountea Inventory Template Editor helps you create and manage item templates for Unreal Engine
              inventory systems. Here's how to get started:

            
    
              
  • Create New Template: Click "New Template" to start creating an item
    • 
              
  • Fill Required Fields: Item Name, Display Name, and Item ID are required
    • 
              
  • Add Assets: Upload icon images and 3D mesh files for your items
    • 
              
  • Export: Export single items (.mnteaitem) or multiple items (.mnteaitems)
    • 
            

          

        


        

          
📝 Field Explanations

          

            

              
Basic Information

              
Item Name: Internal identifier (max 24 chars, no spaces, letters/underscores only)

              
Display Name: User-friendly name shown in the game

              
Item ID: Unique GUID identifier for the item

              
Tooltip Descriptions: Text shown to players in tooltips and inventory

              
Descriptions: Text shown to players in inventory

            


            

              
Properties

              
Item Type: Category for organization and filtering

              
Rarity: Determines item color and perceived value

              
Stack Size: Maximum items that can stack together

              
Weight: Affects inventory capacity calculations

              
Value: Base monetary worth of the item

              
Durability: Maximum condition before item breaks

            


            

              
Item Flags

              
Stackable: Can multiple items occupy one slot?

              
Droppable: Can players drop this item?

              
Usable: Can players use/consume this item?

              
Equippable: Can this item be equipped?

              
Tradeable: Can players trade this item?

              
Quest Item: Is this tied to quest progression?

            

          

        


        

          
🎨 Visual Assets

          

            
Icon: Small image shown in inventory (PNG, JPG, BMP - max 5MB)

            
Mesh: 3D model file for the item (FBX, OBJ - max 50MB)

            
Material Path: Unreal Engine path to material assets

          

        


        

          
⚙️ Equipment System

          

            
When "Equippable" is checked, you can specify which equipment slot the item uses:

            
    
              
  • Helmet, Chest, Legs, Boots, Gloves
    • 
              
  • Main Hand, Off Hand (weapons, shields)
    • 
              
  • Ring, Necklace (accessories)
    • 
            

          

        


        

          
🔧 Custom Properties

          

            
Add custom key-value pairs for special item behaviors:

            
    
              
  • Damage values for weapons
    • 
              
  • Armor ratings for protective gear
    • 
              
  • Spell effects for magical items
    • 
              
  • Any game-specific data you need
    • 
            

          

        


        

          
⌨️ Keyboard Shortcuts

          

            

              Ctrl + S
              Save current template
            

            

              Ctrl + N
              Create new template
            

            

              Ctrl + D
              Duplicate current template
            

            

              Escape
              Close preview panel or current template or help panel
            

          

        


        

          
📤 Export Options

          

            
    
              
  • Export Item (.mnteaitem): Export current template with assets for Unreal Engine
    • 
              
  • Export Items (.mnteaitems): Export multiple templates with assets
    • 
            

          

        


        

          
💡 Tips & Best Practices

          

            
    
              
  • Use descriptive Item Names for better organization
    • 
              
  • Keep icon images small (256x256 or 512x512 recommended)
    • 
              
  • Use consistent naming conventions across your project
    • 
              
  • Preview templates before exporting to catch errors
    • 
              
  • Backup your templates regularly using the export buttons
    • 
              
  • Use selection tools to manage multiple templates efficiently
    • 
            

          

        


        

          
🆘 Troubleshooting

          

            
File Upload Issues: Ensure files meet size and format requirements

            
Validation Errors: Check that all required fields are filled correctly

            
Export Problems: Make sure JSZip library loads properly for file exports

            
Performance: Large numbers of templates may slow down the interface
